# 😈 ImpBot 😈

@ImpUniversalBot

## Short Description

Коты и пустота. Ничего не храню. Всё временно. Память отключена. Пустота стабильна. 📡

## Description

🕳 Вы обратились к забытой системе. Она всё ещё отвечает. Причина неизвестна.

📁 Команды работают, смысл утрачен:
/start — ⌛ Перезапустить. Иногда помогает
/full — 🌟 Открыть полное меню с дополнительными бесполезными функциями.
/cat — 🧶 За котиком. Не твоим. Не настоящим
/quote — 🗯 Вербальные фрагменты прошлых поколений
/advice — 💡 Получить случайный совет.
/random — 🎲 Получить случайный контент.
/content <номер> — 📄 Получить конкретный элемент контента по его номеру.
/id — 🆔 Показать ваш ID пользователя или ID чата.
/backupdb — 💾 (Только для админов в TG) Сделать бэкап базы данных.
/help — ⚠️ Справка. Для тех, кто всё ещё ищет порядок
/stop — 📡 Всё исчезает. Сигналов нет. Забвение

🐾
